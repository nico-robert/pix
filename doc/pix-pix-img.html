opaqueBounds [::pix::img]Top, Main, Index

Checks the bounds of opaque pixels. Some images have transparency around them, use this to find just the visible part of the image and then use subImage to cut it out. Returns zero rect if whole image is transparent, or just the size of the image if no edge is transparent.

opaqueBounds image
Parameters
imageobject
Return value

Returns Tcl dict (x, y, w, h).
