APITop, Main, Index

Currently API tested and supported are :

contextThis namespace provides a 2D API commonly used on the web.
fontThis namespace allows you to write text, load fonts.
imageCrop, resize, blur image and much more.
paintThis namespace plays with colors.
pathsVector Paths.
svgParse, render SVG (namespace pretty limited)
